A Python-based quiz application with user authentication (login/signup), multiple question categories, and predictive analytics for GPA/IQ analysis.

---

## ✨ Features

- 🔐 **User Authentication** - Secure login and signup system
- ❓ **Quiz Engine** - Multiple-choice question system
- 📊 **GPA Prediction** - Linear regression model for GPA forecasting
- 🧠 **IQ Analysis** - Predictive analytics based on study patterns
- 🖥️ **Tkinter GUI** - Clean and intuitive desktop interface
- 📈 **Data-Driven Insights** - Analytical models powered by scikit-learn

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- pip package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/soudasMukherjee/a-quiz-learning-app-with-GPA-IQ-prediction-features.git

# Navigate to project directory
cd a-quiz-learning-app-with-GPA-IQ-prediction-features

# Run the application
python login.py
```

## 📁 Project Structure

```
├── login.py              # Login authentication
├── signup.py           # Signup/registration
├── quiz_brain.py       # Quiz engine logic
├── question_model.py   # Question model class
├── ui.py               # Main UI interface
├── prediction.ipynb    # GPA/IQ prediction models

└── images/             # UI assets
```

## 📊 Prediction Models

The project includes machine learning models built with scikit-learn:
