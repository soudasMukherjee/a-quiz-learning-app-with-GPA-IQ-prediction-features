# SmartQuiz

A Python-based quiz application with user authentication (login/signup), multiple question categories, and predictive analytics for GPA/IQ analysis.

## Features

- User authentication system (login & signup)
- Quiz engine with multiple-choice questions
- GPA & IQ prediction models
- Tkinter-based GUI
- Data-driven insights

## Tech Stack

- Python
- Tkinter
- Pandas
- Jupyter Notebooks

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/soudasMukherjee/a-quiz-learning-app-with-GPA-IQ-prediction-features.git
   ```

2. Run the app:
   ```bash
   python login.py
   ```

## Project Structure

- `login.py` / `signup.py` - Authentication
- `quiz_brain.py` - Quiz engine
- `question_model.py` - Question model
- `ui.py` - GUI interface
- `prediction.ipynb` - GPA/IQ prediction models
- `data.py` - Question data
